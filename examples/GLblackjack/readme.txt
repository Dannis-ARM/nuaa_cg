This is a simple version of the game Blackjack.

Keyboard
"B"  place bet, start's next game with first bet.
"D"  Deal cards, must place bet first.
"H"  Hit add's one card to your deck, must deal first before hiting.
"S"  Stand, take no more cards from dealer.
"N"  Reset's game.
"ESC" exits program.
